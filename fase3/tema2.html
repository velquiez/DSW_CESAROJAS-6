CSS Grid

Al crear un sitio web, gran parte del trabajo consiste en disponer correctamente los distintos elementos que la componen. El diseño debe ser atractivo, pero claro al mismo tiempo, y se debe comprender de forma intuitiva. Las "hojas de estilo en cascada" (CSS - Cascading Style Sheets) constituyen una herramienta para diseñar sitios web con esas características. Mientras que HTML muestra el contenido de forma rudimentaria, CSS es apto para diseños complejos.

Mientras que HTML da la estructura del sitio web en diseño, CSS es un lenguaje que proporciona código para dar estilo a las etiquetas HTML como tipos de letra, colores, tamaños, resaltados, colores de fondo, alineación, entre muchas más aplicaciones.

Técnicas para aplicar CSS3 a HTML:

  1. Incrustado o interno
  2. En línea
  3. Externo

Dentro de la cabecera "head" de una página HTML se agrega la etiqueta "style", la cual permite enseguida agregar las funcionalidades de cambio de color, tamaño, etc.
